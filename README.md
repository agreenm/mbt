# INSTALLING MBT ON BORWSER 
1. INSTALL GOOGLE CHROME BROWSER
2. DOWNLOAD MBT IN GIT
3. GOTO GOOGLE CHROME EXTENSION MANAGEMENT. chrome://extensions/
4. click load unpacked then select mbt.
